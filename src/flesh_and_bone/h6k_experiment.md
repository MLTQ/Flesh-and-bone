# h6k_experiment.py

## Purpose

Evaluates the prepared Kimodo weight-shift cycle with the three frozen H5U MLP
checkpoints and the H6C rule fitted only on the original walk.

## Components

### `H6KConfig` / `load_kimodo_cycle`
- **Does**: Records accepted H5U assets/checkpoints and validates the portable
  versioned 58-phase LBS/bone artifact.

### `bridge_acceptance`
- **Does**: Applies the predeclared humanoid confidence, mapped-role, root-scale,
  identity-skin, finite-state, and horizontal-travel gates without consulting
  flesh results.

### `_render_rules`
- **Does**: Writes aligned teacher, seed-7 MLP, and constitutive five-phase
  contact sheets from each rule's tenth cycle.

### `run_h6k`
- **Does**: Converges one external-motion teacher; evaluates all frozen MLPs
  with feature shift and neighbor ablation; refits H6C only on the original
  walk; evaluates it on Kimodo; and emits three separate verdicts.

## Contracts

| Dependent | Expects | Breaking changes |
|---|---|---|
| H6K ledger | Bridge, MLP, and constitutive verdicts remain separate | Blended score |
| H5U comparison | Checkpoints and normalization are immutable | Kimodo refit |
| H6C comparison | Constitutive coefficients see original walk only | External-motion leakage |
| Visual evidence | Every render is from cycle ten at matching Kimodo phases | Horizon mismatch |
