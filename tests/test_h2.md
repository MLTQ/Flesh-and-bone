# test_h2.py

## Purpose

Provides CPU-fast H2 contracts for discriminating anatomy, destructive wound
semantics, replacement lineage, consecutive local maturation, and wound-aware
metrics.

## Contracts

| Dependent | Expects | Breaking changes |
|---|---|---|
| H2 morphology | Seven nonempty regions; bulb/pad exceed thin-limb radius | Envelope/region definitions |
| H2 wound protocol | Removal clears state and re-feeding records generation 1 | Reservoir mutation semantics |
| H2 maturation | Stability must be consecutive before lock | Counter/lock behavior |
| H2 metrics | Perfect anatomy scores 1 and the wound produces a local drop | Coverage/wound thresholds |
